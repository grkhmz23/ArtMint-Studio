# ArtMint Studio – Demo Recording Plan

## Overview
3-minute demo showing the full flow: prompt → generate → pick → mint → list.

## Suggested Recording Flow

### Scene 1: Landing (10s)
- Open http://localhost:3000
- Show the landing page with "Generate / Mint / Sell" cards
- Click "Open Studio"

### Scene 2: Prompt + Generate (30s)
- Type: "Cosmic ocean waves with aurora borealis"
- Click "Jazz Noir" preset
- Click "Generate 12 Variations"
- Wait for grid to populate
- Slowly scroll through the 12 variations

### Scene 3: Explore Variations (30s)
- Click on 2-3 different variations
- Show the detail panel on the right:
  - Params JSON (read-only)
  - Palette swatches
  - Tags
- Click "More like this" to generate nearby variants
- Show the new grid

### Scene 4: Connect Wallet + Mint (40s)
- Click wallet button (top right)
- Connect Phantom on mainnet
- Select a favorite variation
- Click "Mint this"
- Show the mint preparation (asset upload)
- Navigate to the asset page

### Scene 5: Asset Page + Provenance (40s)
- Show the PNG preview
- Click "Live Render" to toggle to the HTML artifact
- Scroll to the Provenance panel:
  - Show: prompt, template, seed, renderer version, hash, palette
- Click "Copy params" — paste into a text editor to show the canonical JSON
- Click "Re-render 4K" — show the HTML artifact opening in a new tab
- Click the 4K button in the artifact, then "Export PNG"

### Scene 6: List for Sale (20s)
- Enter a price (e.g., 0.5 SOL)
- Click "List Buy Now"
- Show the listing status update

### Scene 7: Profile (10s)
- Navigate to /profile/[wallet]
- Show the grid of minted items with listing status
- End

## Tips
- Use a funded mainnet wallet (no airdrops on mainnet)
- Have the AI API key configured before recording
- Use a clean browser profile for a fresh wallet state
- Record at 1080p or higher
